Semantic UI Installation


   To use Semantic UI as your front-end, you have to import the external link for Semantic UI CSS file, also known as the Semantic UI cdn, in the head tag.

After adding the Semantic UI cdn tags in your head tag, you are good to go and start working on your html using Semantic UI.

For more information on how to use Semantic UI, check out our Tutorial page.



DJango Installation on Windows


Step 1:   If you dont have python downloaded, then download it here, else move to step 2.

   If you are not sure if you have python downloaded or not, then simply type "python --version" in your command prompt.



Step 2:   Make a folder of your name choice.



Step 3:   Open command prompt from your windows and then use it to locate yourself to the directory you made using the "cd" command. Once you are in the folder, then type the command "python -m venv env". Then again, type the command "env\Scripts\activate" as shown in the picture below. You will know you have activated the django environment once you see the "(env)"



Step 4:   Type "pip install django" in the command prompt to download django.



Once you download django, You have to make a new django project and a new django app

Step 5:   To make a new django project: type "django-admin startproject 'project-name-of -our-choice'" in the command prompt as shown below.



Step 6:   Then type "python manage.py startapp 'app-name-of -our-choice'" in the command prompt to make a new django app.
